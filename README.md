# LangBot pansou

LangBot Plugin for Cloud Drive Resource Search, new!

# Usage

This plugin is deployed using Docker. The docker-compose.yml file contains the plugin configuration, which you can modify as needed.

For more details, refer to the official PanSou API:https://github.com/sheetung/pansou

```bash
运行
# Download the configuration file
curl -o docker-compose.yml https://raw.githubusercontent.com/fish2018/pansou/refs/heads/main/docker-compose.yml

# Start the service
docker-compose up -d

# Access the service
http://localhost:8888
```

Select the relevant settings on the configuration page and debug for the fastest request channels.

Trigger by entering keywords in private chat:盘搜 <keywords>

Example:盘搜 蜡笔小新
